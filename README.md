# 🎨 Random Pixel Image Generator

A simple Python script that generates a random RGB image pixel-by-pixel.

## 🚀 Features
- Custom resolution (editable in code)
- Fully random RGB colors
- Saves output as PNG
- Displays the image instantly

## 📦 Requirements
- Python 3.x
- Pillow
- Matplotlib

Install dependencies:
```bash
pip install pillow matplotlib
▶️ Usage
Open the script
Change resolution:
Python
width, height = 500, 500
Run:
Bash
python main.py
🖼 Output
Creates: random_image.png
Displays image on screen
📌 Notes
input() is not used for compatibility with mobile environments (e.g. Pydroid 3)
